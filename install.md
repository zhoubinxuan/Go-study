# Go入门第一天（一）

## windows安装
### 安装Go  
注意环境变量即可
- GOROOT  Go的SDK位置
- GOPATH  Go的开发目录
- PATH    添加Go的bin目录

安装Goglang IDE

测试

2. 创建一个testGo.go文件

```

    fmt.println("Hello Golang!")
}
```
3.IDE中Run单个文件testGo.go文件
